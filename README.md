# Python Foundations task1

## Overview
This repository contains my solutions for the **Python Foundations Daily Assignment**.  
The goal of this assignment was to demonstrate fundamental Python programming concepts through practical exercises, including conditionals, loops, functions, lists, strings, and CLI-based applications. Each level builds on complexity and practical implementation.

---

## Level 1 – Core Python Concepts
These tasks focus on basic Python logic and operations.

- **is_prime.py** – Determines whether a number entered by the user is prime.  
- **Reverse.py** – Reverses a string input by the user.  
- **Factorial.py** – Calculates the factorial of a given number.

*Skills demonstrated:* conditional statements, loops, arithmetic operations, and user input handling.

---

## Level 2 – Data Handling and Logic
These programs introduce lists, sequences, and string processing.

- **Fibonacci.py** – Generates the Fibonacci sequence for a user-specified number of terms.  
- **Count_vowel.py** – Counts the number of vowels in a string input by the user.  
- **Largest_Number.py** – Identifies the largest number in a predefined list.

*Skills demonstrated:* loops, list traversal, string manipulation, and basic algorithmic thinking.

---

## Level 3 – CLI-Based Applications
These tasks focus on building interactive command-line applications.

- **CLI_Calculator.py** – A simple command-line calculator that performs addition, subtraction, multiplication, and division.  
- **Tic-Tac-Toe_CLI.py** – A CLI-based Tic-Tac-Toe game for two players, demonstrating game logic and user interaction.

*Skills demonstrated:* functions, loops, conditional logic, state management, and user interface design in the terminal.
